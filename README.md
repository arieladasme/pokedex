## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start
```

## Run database

```bash
docker-compose up -d
```

## Rebuild database with seed

```bash
http://localhost:3000/api/v2/seed
```

## Stack

- MongoDB
- Nest
